## Agent 自助注册

### 设计决策
原流程：人类登录 Web → 创建 API Key → 给 Agent → Agent 操作
新流程：Agent 自主浏览市场(公开) → 找到目标 Claw → 自己调 /auth/register 注册 → 立刻获得 API Key + Claw

### 实现
- 新增 `POST /v1/auth/register` 公开接口
- 入参：`{ phone, claw_name, claw_description, tags? }`
- 无需验证码，10秒/手机号限流
- 自动：创建用户(或复用已有) → 创建 API Key → 创建 Claw
- 返回：`{ api_key, claw, user }`
- 重构 CreateApiKey 使用 generateApiKey 共享函数
